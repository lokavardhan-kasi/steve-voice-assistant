# Steve - Voice Assistant

A wake word triggered voice assistant built with Python that listens for "Steve" and responds to voice commands.

## Features
- Wake word detection ("Steve")
- Speech to text using Google STT
- Offline text to speech using pyttsx3
- Web search via Google
- Wikipedia queries
- Time and date
- Open applications
- Tells jokes

## Setup
1. Clone the repository
2. Create virtual environment: `python -m venv steve_env`
3. Activate: `steve_env\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python main.py`

## Usage
Say **"Steve"** to wake up then say:
- "What time is it"
- "Search Python tutorials"
- "Who is Elon Musk"
- "Tell me a joke"
- "Open notepad"
- "Goodbye"
